# 🚪 Дверной Дом — Магазин премиум дверей

Современный сайт магазина дверей на **Next.js 14** + **TypeScript**.

## 🚀 Быстрый запуск

```bash
# 1. Перейди в папку проекта
cd doors-shop

# 2. Установи зависимости (1-2 минуты)
npm install

# 3. Запусти сайт
npm run dev
```

Открой **http://localhost:3000** — сайт работает! 🎉

## 📦 Что внутри

- ⚡ **Next.js 14** с App Router
- 🎨 **Премиум дизайн** с золотом и тёмной темой
- 🔍 **Каталог** с фильтрацией по категориям
- 📝 **Форма заявки** на замер
- 📱 **Полностью адаптивный** дизайн
- ✨ **Анимации** на CSS
- 🎯 **TypeScript** для надёжности

## 🌐 Деплой на Vercel (бесплатно)

1. Зарегистрируйся на **[vercel.com](https://vercel.com)** через GitHub
2. Создай репозиторий на GitHub и залей туда папку
3. На Vercel: **Add New** → **Project** → выбери репо
4. Нажми **Deploy** — через 2 минуты сайт онлайн!

Получишь ссылку типа: `https://doors-shop.vercel.app`
