# 🚪 Дверной Дом — Магазин премиум дверей

Современный сайт магазина дверей на **Next.js 14** + **TypeScript**.

## 🚀 Быстрый запуск

```bash
# 1. Перейди в папку проекта
cd doors-shop

# 2. Установи зависимости (1-2 минуты)
npm install

# 3. Запусти сайт
npm run dev
```

## 📦 Что внутри

- ⚡ **Next.js 14** с App Router
- 🎨 **Премиум дизайн** с золотом и тёмной темой
- 🔍 **Каталог** с фильтрацией по категориям
- 📝 **Форма заявки** на замер
- 📱 **Полностью адаптивный** дизайн
- ✨ **Анимации** на CSS
- 🎯 **TypeScript** для надёжности

## 🛠️ Технологии

- Next.js 14
- React 18
- TypeScript 5
- CSS Variables
- Cormorant Garamond + Manrope fonts
